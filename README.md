# MP3播放器

比亚迪 DiLink 车机本地音乐播放器，支持 **卡拉 OK 逐字歌词**、**全屏封面播放页**、**桌面悬浮歌词**、**方向盘 / 语音切歌**。

当前版本：**v3.0.1**

## 功能一览

### 播放与列表
- 扫描 SD 卡 / 车机 `Music` 目录 / 自选文件夹中的 MP3 等音频
- 底部三页导航：播放列表 · 播放 · 设置
- 搜索、筛选（全部 / 有歌词）
- 顺序播放 / 随机播放
- 进度条拖动、当前时长 / 总时长显示
- 启动时自动恢复上次播放（可关）

### 歌词
- 自动匹配同名 `.lrc` 文件；无本地歌词时可联网获取（lrclib.net）
- 应用内滚动歌词 + 全局悬浮双行歌词
- 卡拉 OK：已唱字变色，未唱字半透明
- 歌词过长自动换行（最多 1–4 行），不超出屏幕
- 8 种清淡主题：网易云、塞尔达、马里奥、淡粉 / 绿 / 蓝、薄荷、薰衣草
- 可调字体、字号、当前行放大、下一行缩小、平滑滚动等

### 封面
- 播放页全屏专辑封面 + 磨砂玻璃效果
- 优先读取 MP3 内嵌封面，无封面时可联网获取（iTunes）

### 车机操控（v3.0.1 新增）
- **方向盘媒体键**：上一首 / 下一首 / 播放暂停
- **车内语音**：说「上一首」「下一首」「暂停」「继续播放」等，由系统转给当前正在播放的媒体应用；**请先在本 App 里开始播放音乐**，并确保本应用是车机当前媒体焦点
- 通知栏媒体控制（带上一首 / 下一首按钮）

> 语音能否切歌取决于比亚迪 DiLink 是否把指令发给「当前媒体会话」。本 App 已接入标准 MediaSession，与网易云等播放器同一套机制；若语音无效，可在车机设置里确认默认音乐应用或先手动打开本 App 播放一次。

## 音乐与歌词放置

```
/sdcard/Music/
  ├── 歌手A/
  │   ├── 歌曲1.mp3
  │   └── 歌曲1.lrc
  └── 歌曲2.mp3
  └── 歌曲2.lrc
```

也可在 **设置 → 选择文件夹** 添加任意目录（支持 SAF 文档树）。

**LRC 格式支持：**

- 普通行级：`[01:02.34]歌词内容`
- 字级（更准确卡拉 OK）：`[01:02.00]你[01:02.30]好[01:02.60]啊`
- 仅有行级时间时，会自动按字数均分逐字时间

## 编译 APK

### 方式一：GitHub Actions（推荐，无需本地 Android 环境）

1. 将代码推送到 GitHub 仓库
2. 打开 **Actions** 页，手动 **Run workflow**
3. 构建完成后在 Artifacts 中下载 `app-debug.apk`

### 方式二：Android Studio 本地编译

1. 用 **Android Studio** 打开本项目目录 `mp3-player`
2. 等待 Gradle 同步完成
3. 菜单 **Build → Build Bundle(s) / APK(s) → Build APK(s)**
4. APK 路径：`app/build/outputs/apk/debug/app-debug.apk`

## 安装到车机

### U 盘安装

1. 复制 APK 到 U 盘
2. 车机文件管理器打开 U 盘，点击安装
3. 允许安装未知来源（如系统提示）

### ADB 安装

```bash
adb connect <车机IP>   # 若无线调试
adb install -r app-debug.apk
```

## 车机必设权限（DiLink 4.0）

1. **存储权限**：首次打开 App 允许读取音频
2. **显示在其他应用上层**（悬浮歌词关键）
   - 设置 → 应用 → MP3播放器 → 权限
   - 或 App 内「设置 → 开启悬浮窗权限」
3. **极速模式 → 白名单**：加入 MP3播放器，防后台被杀
4. **禁止自启动**：确认没有误关本应用

## 使用步骤

1. 打开 **MP3播放器** → **设置** 中添加扫描目录 → 点击 **扫描音乐**
2. 在 **播放列表** 点歌，或到 **播放** 页控制
3. **设置** 里开启悬浮歌词、选择歌词主题与字号
4. 切到 **桌面** 或 **地图**，可看到悬浮歌词
5. 行驶中可用 **方向盘按键** 或 **语音** 切歌（需本 App 正在播放）

## 项目结构

```
app/src/main/java/com/car/mp3player/
  MainActivity.kt              # 主界面、底部导航
  MusicPlaybackService.kt      # 后台播放、MediaSession、歌词同步
  LyricsOverlayService.kt      # 全局悬浮歌词
  playback/
    PlaybackStateHolder.kt     # 播放状态
    CarPlaylistPlayer.kt       # 方向盘 / 语音切歌转发
  ui/
    PlaylistFragment.kt        # 播放列表
    PlayerFragment.kt          # 全屏封面播放页
    SettingsFragment.kt        # 设置与歌词主题
    ScrollLyricView.kt         # 播放页滚动歌词
    KaraokeLyricView.kt        # 悬浮卡拉 OK 歌词
    LyricRenderer.kt           # 歌词绘制与换行
  lrc/LrcParser.kt             # LRC 解析
  data/                        # 扫描、设置、联网歌词/封面
```
