# MP3播放器

比亚迪 DiLink 车机本地音乐播放器，支持 **卡拉 OK 逐字歌词**、**全屏封面播放页**、**桌面/仪表屏悬浮歌词**、**方向盘 / 语音切歌**。

当前版本：**v3.1.0**

## 功能一览

### 播放与列表
- 扫描 SD 卡 / 车机 `Music` 目录 / 自选文件夹中的 MP3 等音频
- 底部三页导航：播放列表 · 播放 · 设置
- 搜索、按歌手浏览、按歌名/时长排序
- 点进歌手后只播放该歌手的歌曲
- 顺序播放 / 随机播放
- 进度条拖动、当前时长 / 总时长显示
- **开机自动启动并续播**（需开启设置）
- **启动时自动恢复播放**

### 歌词
- 自动匹配同名 `.lrc` 文件；无本地歌词时自动联网获取（lrclib.net，匹配更准确）
- **长按播放页歌词**可修改歌名/歌手并重新搜索
- 应用内滚动歌词 + 全局悬浮双行歌词 + **仪表屏滚动歌词**（需车机有副屏）
- 卡拉 OK：已唱字变色，未唱字半透明
- 歌词过长自动换行（最多 1–4 行），不超出屏幕
- 8 种清淡主题：网易云、塞尔达、马里奥、淡粉 / 绿 / 蓝、薄荷、薰衣草
- 可调字体、字号、当前行放大、下一行缩小、平滑滚动等

### 封面
- 播放页全屏专辑封面 + 磨砂玻璃效果
- 优先读取 MP3 内嵌封面，无封面时可联网获取（iTunes）

### 车机操控
- **方向盘媒体键**：上一首 / 下一首 / 播放暂停
- **车内语音**：说「上一首」「下一首」等（需本 App 正在播放）
- 通知栏媒体控制

## 音乐与歌词放置

```
/sdcard/Music/
  ├── 歌手A/
  │   ├── 歌曲1.mp3
  │   └── 歌曲1.lrc
  └── 歌曲2.mp3
  └── 歌曲2.lrc
```

也可在 **设置 → 选择文件夹** 添加任意目录。

**歌词不对时：** 在播放页 **长按歌词区域**，修改歌名/歌手后点「搜索」重新下载。

## 编译 APK

### GitHub Actions（推荐）

1. 推送代码到 GitHub
2. **Actions** 页手动 **Run workflow**
3. 下载 `app-debug.apk`

### Android Studio 本地编译

`app/build/outputs/apk/debug/app-debug.apk`

## 安装到车机

U 盘安装或 `adb install -r app-debug.apk`

## 车机必设权限（DiLink 4.0）

1. **存储权限**：允许读取音频
2. **显示在其他应用上层**：悬浮歌词
3. **极速模式白名单**：防后台被杀
4. **开机自启动**（如车机有该设置）：允许本应用开机启动
5. 开启 **开机自动启动并续播** + **启动时自动恢复播放**

## 使用步骤

1. **设置** 中添加扫描目录 → **扫描音乐**
2. **播放列表** 可按歌手浏览，或按时长排序
3. 点歌播放，长按歌词可重新搜索
4. 开启悬浮歌词 / 仪表屏歌词（仪表屏需车机支持副屏显示）
