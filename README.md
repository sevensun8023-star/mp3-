# MP3播放器

比亚迪 DiLink 车机本地音乐播放器，支持 **黑胶唱片播放页**、**卡拉 OK 逐字歌词**、**桌面/仪表屏悬浮歌词**、**方向盘 / 语音切歌**、**开机续播**。

当前版本：**v3.7.1**

---

## 功能一览

### 播放与列表

- 扫描 SD 卡 / 车机 `Music` 目录 / 自选文件夹中的 MP3 等音频
- **播放列表缓存**：扫描结果自动保存，再次打开立即显示
- 底部三页导航：播放列表 · 播放 · 设置（**打开 App 默认进入播放页**）
- 搜索、按歌手浏览、按歌名/时长排序；点进歌手后只播放该歌手歌曲
- **默认随机播放**（可切换顺序播放，会记住你的选择）
- 进度条拖动、当前时长 / 总时长显示
- **开机自动续播**（后台恢复，可选回到桌面，不强制打开界面）
- **启动问候音**：「hello，林俊」（设置里可关）
- 双播放器冲突修复：切歌、方控、续播更稳定

### 歌词

- 自动匹配歌曲同目录的同名 `.lrc` 文件
- **联网歌词自动保存到歌曲旁边**（`歌曲名.mp3` → `歌曲名.lrc`），卸载 App 后歌词仍在
- 旧版 App 内部缓存的歌词，播放时会**自动迁移**到歌曲目录
- **多源歌词搜索**（国内优先）：网易云 → 酷狗 → QQ 音乐 → LrcLib
- **长按播放页歌词**可修改歌名/歌手并重新搜索
- 播放页：点黑胶唱片切换歌词视图；歌词可**上下滑动**查看作词/作曲等信息
- 应用内滚动歌词 + 全局悬浮双行歌词 + **仪表屏滚动歌词**（需车机有副屏）
- 卡拉 OK：已唱字高亮，未唱字半透明
- **悬浮歌词配色**参考网易云：正在唱红色，未播放半透明白
- 歌词字体：系统默认、思源黑体、站酷快乐体、马善政楷、思源宋体（需联网首次下载字体）
- 歌词配色主题：经典白、网易云红、淡粉 / 绿 / 蓝、薄荷、薰衣草
- 可调字号、当前行放大、下一行缩小、平滑滚动、悬浮加粗等

### 封面与播放页

- **黑胶唱片**旋转封面（唱片与封面一起转），点击切换歌词
- 播放页背景随专辑封面取色；底部导航与播放页色调融合
- 优先读取 MP3 内嵌封面，无封面时可联网获取（iTunes）

### 外观主题

- **应用主题**：网易云、塞尔达、马里奥
- **浅色 / 深色 / 跟随系统**
- 歌词配色与应用主题独立

### 车机操控

- **方向盘媒体键**：优先由本 App 接管上一首 / 下一首 / 播放暂停
- **车内语音**：说「上一首」「下一首」等（需本 App 正在播放）
- 通知栏媒体控制；MediaSession 与音频焦点增强，减少误切到车机自带播放器

## 音乐与歌词放置

```
/sdcard/Music/
  ├── 歌手A/
  │   ├── 歌曲1.mp3
  │   └── 歌曲1.lrc      ← 可自己放，也可由 App 联网后自动创建
  └── 歌曲2.mp3
  └── 歌曲2.lrc
```

也可在 **设置 → 选择文件夹** 添加任意目录。

**没有本地 LRC 时：** 开启「联网获取歌词」，App 会自动搜索并**保存 `.lrc` 到歌曲同目录**。

**歌词不对时：** 在播放页 **长按歌词区域**，修改歌名/歌手后点「搜索」重新下载（会覆盖同目录 `.lrc`）。

## 编译 APK

### GitHub Actions（推荐）

1. 推送代码到 GitHub（文件放在 `app/` 目录下，**不要覆盖**仓库根目录的 `build.gradle.kts`）
2. **Actions** 页手动 **Run workflow**
3. 下载 `app-debug.apk`

### Android Studio 本地编译

输出路径：`app/build/outputs/apk/debug/app-debug.apk`

## 安装到车机

U 盘安装或 `adb install -r app-debug.apk`

## 车机必设权限（DiLink 4.0）

1. **存储权限**：允许读取（及写入）音频目录，以便保存 `.lrc`
2. **显示在其他应用上层**：悬浮歌词
3. **自启动 / 后台运行白名单**（极速模式）：防开机续播被杀
4. 开启 **开机自动启动**、**启动时自动恢复播放**、**启动问候音**（按需）

---

## 使用步骤

1. **设置** 中添加扫描目录 → **扫描音乐**
2. 打开 App 即在 **播放页**；默认 **随机播放**
3. 点黑胶唱片看歌词，手指上下滑可浏览整首歌词
4. **设置 → 歌词配色 / 字体** 调整样式；**设置 → 外观** 换应用主题
5. 开启悬浮歌词 / 仪表屏歌词（仪表屏需车机支持副屏）
